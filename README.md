# Future Bird

**Future Bird** es un juego de supervivencia inspirado en *Flappy Bird*, desarrollado en **Python** utilizando **Pygame**.

El objetivo es mantenerse en el aire el mayor tiempo posible mientras esquivas obstáculos y enemigos generados de forma procedimental. A medida que juegas, tu energía se recarga automáticamente para usar la habilidad **Dash**, la cual te permite destruir amenazas y aumentar tus posibilidades de supervivencia.

---


## Importante

Para que el juego funcione correctamente, asegúrate de descargar **todos los archivos del proyecto**, incluyendo las imágenes y los archivos de audio.

---

## Instalación

Instala la única dependencia del proyecto:

```bash
pip install pygame
```

---

## ▶️ Ejecución

Desde la carpeta del proyecto, ejecuta:

```bash
python main.py
```

---

## 🎮 Controles

| Acción | Teclado | Ratón |
|---------|----------|--------|
| Saltar | `Espacio` | `Clic izquierdo` |
| Dash | `Shift` o `D` | `Clic derecho` |
| Iniciar / Reintentar | `Espacio` | `Clic izquierdo` |
